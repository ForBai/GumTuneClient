# GumTuneClient for Minecraft 1.21.8 (Fabric)

This is the Fabric 1.21.8 version of GumTuneClient.

## Building

Ensure you have Java 21 installed, then run:

```bash
./gradlew build
```

The compiled mod will be in `build/libs/gumtuneclient-0.7.6-beta3-fabric.jar`

## Installation

1. Install Minecraft 1.21.8
2. Install Fabric Loader 0.16.10+
3. Download Fabric API 0.111.0+ for 1.21.8
4. Place both the Fabric API and this mod in your `.minecraft/mods` folder

## Current Status
